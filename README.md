# 幸福感知器

基于塞利格曼 PERMA 模型的每日幸福事件记录器。月历方格中，每条幸福记录是一个会随手机重力摇摆、触边微回弹的动画图标。

- **零服务器**：数据存手机本地（localStorage），可离线使用（PWA）
- **可安装**：手机浏览器「添加到主屏幕」即可当 App 用
- **PERMA 五维**：积极情绪 / 沉浸心流 / 达成成就 / 人际关系 / 超越意义

## 本地运行

```bash
python -m http.server 8000
# 浏览器打开 http://localhost:8000
```

> 重力感应在 iOS Safari 上需要 HTTPS。本地用 localhost 可调试日历与记录流程；真机重力体验请通过 GitHub Pages（自带 HTTPS）访问。

## 技术

纯静态：HTML + CSS + 原生 JS，自写轻量粒子物理引擎（重力积分 + 触边回弹），无构建步骤、无第三方依赖。
